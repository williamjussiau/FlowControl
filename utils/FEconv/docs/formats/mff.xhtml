1. Description

The MFF (Modulef Formatted Field) is a field format associated with the MFM (Modulef Formatted Mesh).

The main features of this format are:

  • It can only store a single field.
  • It contains the number of values and a list of values.
  • It can store fields at nodes or on elements.
  • It can only store values over the whole mesh.
  • Support scalar and vector fields with real values.
  • All values are given by double precission floats.

2. Features supported by feconv

  • The number of field values must be a multiple of the number of nodes or elements. The ratio between these quantities indicates the number of field components.
  • In order to mantain the consistence, the ordering of node coordinates must agree among the field and the mesh data.
